Ada Lovelace (edition 2019)

Description of the projet :

The challenge Ada Lovelace is an computer science challenge where girls from high school have to create a web site around a topic in a group of 4-5. This is project aim to promote the place of women in the numeric world, which is often consider to be man only has shown by statistics :

There is not even 1/3 of women in engineer formation (first line). In order to change things, the Lycée Le Dantec, the ENSSAT, Orange and Elles Bougent decide to create this event to touch young girls and give them a good first experience of the numeric world.

To achieve this goal, groups are helped by women who are in higher study, this help is mandatory since the schoolgirl have no notion in web. But the tutor also give them an example of woman that work in the numeric, which can break stereotypes.

My contribution to this project :

As a student at ENSSAT Lannion, I have the opportunitie to participate to projects that aim to create the student life. I have been the coordinator of this project. As explained above, this project have a lot of actors, my goal was to coordinate these actors in order to create a good challenge for the students.

For this, I had to manage the student team and help them, I have participate in the creation of the budget for the event, to the sponsoring research and some logistic details. But I also had to work with the others actor and communicate with them, organizing meeting.

This project is the first one that made me work with so many differents person, it was really challenging to do it along side with the Hackathon ENSSAT and my 2nd year at the ENSSAT Lannion. I am really proud to have contribute to it and of the success of this edition.
